# Creating an Account in TradeZella

A TradeZella Account refers to a trading account (not a user account) that tracks trades for a specific broker, account number, or trading strategy. You can create multiple accounts to track different brokers, sub-accounts, or trading styles.

## What is a TradeZella Account?

A TradeZella Account allows you to:

- Track trades from a specific broker or brokerage account
- Separate trading activity by asset class (stocks, options, futures, forex)
- Maintain distinct P&L records for different strategies
- Import trades from various brokers independently
- Monitor performance metrics separately for each account

## How to Create a New Account

### Step 1: Access Account Creation

1. Log into your TradeZella dashboard
2. Click the **"Accounts"** tab in the main navigation
3. Click **"Create New Account"** or the **"+"** button
4. The account creation wizard will appear

### Step 2: Enter Account Details

Fill in the following information:

**Account Name**: Give your account a descriptive name (e.g., "Primary Schwab Account", "Futures Trading", "Options Strategy")

**Broker**: Select your broker from the dropdown list. Popular options include:
- Charles Schwab / Thinkorswim
- Interactive Brokers
- Tradovate
- TD Ameritrade
- Questrade
- Other brokers via manual import

**Account Type**: Specify the type of trading:
- Cash Account
- Margin Account
- Futures Account
- Options Account

### Step 3: Select Asset Class

Choose the primary asset class for this account:

- **Stocks**: For equity trading
- **Options**: For options trading
- **Futures**: For futures contracts
- **Forex**: For foreign exchange trading
- **Multi-Asset**: For accounts trading multiple asset types

### Step 4: Set Initial Balance

Enter your starting account balance. This should be the initial deposit or account value when you begin tracking trades in TradeZella. You can adjust this balance later if needed.

### Step 5: Configure Trade Settings

Choose your profit calculation method:
- **FIFO** (First In, First Out) - Default
- **LIFO** (Last In, First Out)
- **Weighted Average**

### Step 6: Complete Setup

1. Review all entered information
2. Click **"Create Account"**
3. Your account is now created and ready to track trades

## After Creating Your Account

Once created, you can:

- Import historical trades from your broker
- Manually add trades
- Set up automated sync with your broker
- Configure commissions and fees
- Create trade tags for organization
- View performance reports and analytics

## Managing Multiple Accounts

You can switch between accounts in the **Accounts** section. Each account maintains independent:

- Trade history
- Performance metrics
- Balances and P&L
- Settings and preferences
- Import history
