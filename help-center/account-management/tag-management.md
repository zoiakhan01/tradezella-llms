# Tag Management in TradeZella

Tags are custom labels you assign to trades for easy categorization and organization. Use tags to organize trades by strategy, market condition, setups, emotion, or any criteria meaningful to your analysis. TradeZella supports both individual tags and hierarchical tag categories.

## Why Use Tags

Tags help you:

- **Organize Trades**: Group similar trades for analysis
- **Strategy Tracking**: Label trades by strategy used
- **Performance Analysis**: Filter and compare specific trade types
- **Identify Patterns**: Find successful and unsuccessful setups
- **Emotional Tracking**: Mark trades influenced by emotion
- **Market Conditions**: Label trades by market environment
- **Setup Categories**: Tag specific technical patterns

## Creating Tags

### Individual Tags

To create a new tag:

1. Go to your **"Accounts"** section
2. Navigate to **"Tags"** or **"Tag Management"**
3. Click **"Create New Tag"** or **"+"** button
4. Enter a tag name (e.g., "Breakout", "Support Bounce")
5. Optionally add a description
6. Choose a color for visual identification
7. Click **"Create"**

### Tag Categories

Create organized groups of related tags:

1. In **Tag Management**, click **"Create New Category"**
2. Enter a category name (e.g., "Setups", "Market Conditions", "Strategies")
3. Optionally add a category description
4. Click **"Create Category"**
5. Add tags to the category by:
   - Creating new tags within the category
   - Or assigning existing tags to it

**Example Tag Structure**:
- **Setups** Category
  - Support Bounce
  - Breakout
  - Pullback
  - Reversal
- **Market Conditions** Category
  - Trending
  - Range-bound
  - Volatile
  - Calm

## Assigning Tags to Trades

### Via Trade Journal

1. Open your **"Trade Journal"** or **"Trades"** section
2. Find the trade you want to tag
3. Click the **tag icon** or **"Add Tag"** button on the trade
4. Select from existing tags or create a new tag
5. Multiple tags can be assigned to one trade
6. Click **"Save"** to apply tags

### While Entering a Trade

When manually entering a trade:

1. In the trade entry form, look for **"Tags"** field
2. Start typing a tag name
3. Select from suggestions or create a new tag
4. Add multiple tags if desired
5. Complete the trade entry
6. Tags save with the trade

### Bulk Tagging

Tag multiple trades at once:

1. Select multiple trades in your journal (checkboxes)
2. Click **"Bulk Actions"** or **"Edit Multiple"**
3. Choose **"Add Tags"**
4. Select or create tags to apply to all selected trades
5. Click **"Apply"**

## Managing Tags

### Viewing All Tags

To see all tags in your system:

1. Go to **"Tag Management"** in your account settings
2. See a complete list of all tags
3. View tag descriptions
4. See how many trades use each tag
5. View tag colors

### Editing Tags

To modify a tag:

1. In **Tag Management**, find the tag
2. Click **"Edit"** or the tag name
3. Change the tag name, description, or color
4. Click **"Save Changes"**

**Note**: Changes apply to all trades with that tag.

### Deleting Tags

To delete a tag:

1. Go to **"Tag Management"**
2. Find the tag to delete
3. Click **"Delete"** or the trash icon
4. Confirm deletion
5. You'll be asked what to do with trades using that tag:
   - Remove the tag from affected trades
   - Delete the trades too
   - Move trades to a different tag

### Renaming Tag Categories

To rename a category:

1. In **Tag Management**, find the category
2. Click **"Edit"** next to the category name
3. Enter the new name
4. Click **"Save"**

All tags in the category remain unchanged.

## Using Tags for Analysis

### Filtering by Tags

Filter your trade journal by tags:

1. In your **"Trade Journal"**
2. Find the **"Filter"** or **"Search"** options
3. Select a tag from the dropdown
4. Journal shows only trades with that tag
5. Select multiple tags to show trades with ANY of those tags

### Performance by Tag

View performance metrics for specific tags:

1. Go to **"Reports"** or **"Analytics"**
2. Look for **"Performance by Tag"** or similar report
3. See win rate, profit, and other metrics broken down by tag
4. Analyze which strategies or setups are most profitable
5. Identify areas for improvement

### Tag-Based Grouping

Group your trades by tags:

1. In your trade journal, select **"Group By"** > **"Tags"**
2. Trades are organized by tag
3. Each tag shows all associated trades
4. See totals and metrics per tag group

## Tag Organization Best Practices

### Naming Conventions

- **Clear Names**: Use descriptive names (not abbreviations)
- **Consistency**: Use same naming for similar concepts
- **Avoid Duplicates**: Don't create similar tags like "Breakout" and "Break Out"
- **Action-Oriented**: Consider tagging by action ("FOMOed", "Panic Sold")

### Suggested Tag Categories

**By Strategy**:
- Support Bounce, Resistance Break, Trend Following, Mean Reversion

**By Market Condition**:
- Bullish, Bearish, Consolidation, High Volatility

**By Emotion**:
- Confidence, Doubt, Fear, FOMO, Revenge Trading

**By Performance**:
- Best Setup, Worst Setup, Lucky Win, Painful Loss

**By Outcome**:
- Planned Exit, Early Exit, Late Exit, Liquidated

## Tips for Effective Tagging

1. **Tag Consistently**: Tag trades as you enter them, not later
2. **Be Specific**: Use detailed tags for better analysis
3. **Limit Tags**: Don't create too many tags (aim for 20-30)
4. **Regular Review**: Periodically review and clean up unused tags
5. **Use Categories**: Organize tags into logical groups
6. **Color Code**: Use colors for quick visual identification
7. **Document Meanings**: Add descriptions so you remember why you created tags

## Exporting Tag Data

To export trades with tags:

1. Use your account's **"Export"** feature
2. Export options include CSV, PDF, or Excel
3. Tags appear in the exported data
4. Useful for external analysis or record-keeping
