# Enable Dark Mode

TradeZella offers a dark mode theme to reduce eye strain during extended trading sessions and improve visibility in low-light environments. Dark mode can be easily enabled from your profile settings.

## How to Enable Dark Mode

### Via Profile Settings

1. Click on your avatar or username in the top-right corner of the dashboard
2. Select **"Profile Settings"** from the dropdown menu
3. Look for the **"Theme"** or **"Appearance"** section
4. Select **"Dark Mode"** from the available theme options
5. Changes apply immediately to your account

### Via Settings Panel

Alternatively, you can access dark mode settings from the main Settings area:

1. Click the **Settings** icon (gear icon) in the main navigation
2. Select **"Appearance"** from the left sidebar
3. Toggle **"Dark Mode"** to enable it
4. The entire interface will switch to the dark theme

## Dark Mode Features

Dark mode in TradeZella includes:

- **Dark Background**: Reduces eye strain with a dark UI background
- **Light Text**: All text becomes light-colored for proper contrast
- **Chart Optimization**: Charts are optimized for dark mode viewing
- **Button and Icon Styling**: All interface elements are adapted for dark mode
- **Consistent Experience**: Dark mode applies across all TradeZella pages and features

## System-Based Dark Mode

TradeZella can also automatically detect your system's dark mode preference:

1. Go to **Settings** > **Appearance**
2. Select **"System Default"** to match your operating system's theme setting
3. Your TradeZella theme will automatically switch when your system theme changes

## Switching Back to Light Mode

To return to light mode:

1. Access **Profile Settings** or **Settings** > **Appearance**
2. Select **"Light Mode"** from the theme options
3. The interface will immediately revert to the standard light theme

## Benefits of Dark Mode

- Reduced eye strain during long trading sessions
- Better visibility in dimly lit environments
- Improved battery life on OLED screens
- Professional, modern appearance
- Easier to focus on trading data and charts
