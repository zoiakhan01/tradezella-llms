# Introducing Our New and Updated Text Editor

Discover the enhanced text editor that brings powerful formatting and media capabilities to your trading notes.

## Core Features

The updated text editor provides comprehensive tools for capturing and organizing your trading insights. Key features include:

- **Drag-and-drop images** with custom captions
- **Slash commands** for quick formatting (headings, tables, lists, checklists, quotes, dividers, code blocks, and collapsible sections)
- **Emoji insertion** for visual organization
- **Embedded media** including tweets and YouTube videos
- **PDF export** with integrated trade and day statistics
- **Multi-format export/import** using Lexical format for seamless data portability
- **Speech-to-text support** for hands-free note creation

## Advanced Capabilities

These features work together to create a flexible writing environment that adapts to your trading workflow. Whether you're documenting trade setups or analyzing market conditions, the editor provides the tools you need.
