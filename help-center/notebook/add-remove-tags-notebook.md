# How to Add and Remove Tags

Tags are labels you assign to notes for easy searching and categorization. They complement folders and help you quickly find notes by topic, strategy, or any other criteria.

## Why Use Tags?

Tags provide flexible organization:

- Search notes across all folders using a single tag
- Categorize notes by multiple criteria simultaneously
- Track analysis themes without rigid folder structures
- Filter notes by tag in reports and analytics
- Cross-reference related notes
- Build a personalized tagging system that matches your workflow

## Adding Tags to a Note

To add tags to a new or existing note:

1. Open the Notebook section
2. Create a new note or click to edit an existing note
3. Look for the "Tags" field (typically near the top of the note)
4. Click in the Tags field
5. Type a tag name (e.g., "EUR/USD", "Breakout", "High-Volume")
6. Press Enter to add the tag
7. Repeat to add multiple tags to the same note
8. Save the note

## Creating Tags

Tags are created on-the-fly as you type:

1. Click the Tags field in your note
2. Type a new tag name (no special characters needed)
3. Press Enter to create and add it
4. The tag will be added to your note and available for future use

## Best Practices for Tagging

- **Be Consistent**: Use the same tag names across notes (e.g., always "breakout", not "breakouts")
- **Use Clear Names**: Choose descriptive tags that make sense (e.g., "momentum-trading" vs "mt")
- **Keep it Simple**: Use 2-5 tags per note for optimal organization
- **Combine with Folders**: Use folders for broad categories and tags for specific attributes
- **Review Tags Regularly**: Audit and consolidate similar tags periodically

## Common Tag Examples

Consider these tag categories for your trading notes:

- **Strategy Tags**: "scalping", "swing-trade", "day-trade", "position-trade"
- **Instrument Tags**: "eurusd", "gbpusd", "spy", "qqq"
- **Market Condition Tags**: "trending", "ranging", "volatile", "calm"
- **Technical Tags**: "support-resistance", "fibonacci", "moving-average", "orderflow"
- **Status Tags**: "active-strategy", "testing", "archived", "review-needed"

## Removing Tags from Notes

To remove a tag from a note:

1. Open the note containing the tag
2. Locate the tag in the Tags field
3. Click the "X" button next to the tag name
4. Or click the tag and select "Remove"
5. Save the note

## Viewing All Tags

See all tags you've created:

1. Open the Notebook section
2. Click on "Tags" in the sidebar or navigation menu
3. View your complete tag list
4. Click any tag to see all notes with that tag

## Filtering Notes by Tag

Find notes quickly using tags:

1. In the Notebook section, click on a specific tag
2. All notes with that tag will display
3. Combine with folder view for more specific results
4. Use search bar to find tags by name

## Renaming or Merging Tags

If you want to standardize your tags:

1. Go to Notebook settings or tag management
2. Find the tag you want to modify
3. Select "Rename Tag" or "Merge Tags"
4. Choose a new name or select the tag to merge into
5. Apply changes (this updates all affected notes)

## Tag Analytics

Some TradeZella reports include tag-based analysis:

1. View performance by tag in Reports section
2. See P&L breakdown for each tag
3. Analyze win rate per tag
4. Compare profitability across different tag categories
