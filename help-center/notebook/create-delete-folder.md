# How to Create or Delete a Folder in the Notebook

Organizing your notes into folders helps you maintain a structured and efficient Notebook. TradeZella makes it simple to create, rename, and delete folders.

## Why Use Folders?

Folders help you:

- Organize notes by trading strategy or instrument
- Group notes by time period (daily, weekly, monthly)
- Separate different types of analysis (technical, fundamental, journal)
- Keep your Notebook clean and easy to navigate
- Find notes quickly using folder structure
- Archive old notes without losing access

## How to Create a Folder

To create a new folder in your Notebook:

1. Open the Notebook section from your TradeZella dashboard
2. Look for the "+" button or "New Folder" option
3. Click to create a new folder
4. Enter a descriptive folder name (e.g., "EUR/USD Analysis" or "Daily Charts")
5. Press Enter or click "Create"
6. The new folder will appear in your folder list

## Creating Subfolders

You can create folders within folders for deeper organization:

1. Click on a folder to open it
2. Look for the "New Subfolder" option
3. Name your subfolder
4. Press Enter to create

Example structure:
- Trading Analysis
  - Technical Analysis
  - Fundamental Analysis
  - Trade Setups

## Renaming Folders

To rename an existing folder:

1. Right-click on the folder or click the Menu button (three dots)
2. Select "Rename Folder"
3. Edit the folder name
4. Press Enter or click "Save"

## Moving Notes Between Folders

Organize notes by dragging them:

1. Open the Notebook section
2. Click and drag a note to move it to a different folder
3. Or right-click a note and select "Move to Folder"
4. Choose the destination folder
5. Confirm the action

## How to Delete a Folder

To remove a folder from your Notebook:

1. Locate the folder you want to delete
2. Right-click on it or click the Menu button (three dots)
3. Select "Delete Folder" from the menu
4. Confirm the deletion when prompted

## What Happens When You Delete a Folder?

When deleting a folder, you have options:

- **Delete Folder Only**: Move all notes to parent directory
- **Delete Folder and Contents**: Remove the folder and all notes inside
- **Archive Instead**: Move folder to archive rather than permanently deleting

Choose your preferred action in the confirmation dialog.

## Folder Management Best Practices

- Use clear, descriptive folder names
- Keep folder hierarchy to 3 levels or fewer for easy navigation
- Archive old folders instead of deleting them
- Use consistent naming conventions across folders
- Periodically review and reorganize your folder structure

## Searching Within Folders

Once organized, find notes quickly:

1. Click into the folder you want to search
2. Use the search bar to find specific notes
3. Filter by tags or date created
4. View notes in list or card view
