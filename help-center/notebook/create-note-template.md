# How to Create a Default Template for Adding Notes

Create custom templates to ensure consistency and save time when creating new notes.

## Creating Your Template

Follow these steps to set up a default template for any folder:

### Step 1: Access Template Settings

1. Navigate to **Notebook** section
2. Open the **Daily Journal** folder (or your desired folder)
3. Click the **three-dot icon** next to the folder name

### Step 2: Edit and Customize

1. Select **"Edit Template"** from the menu
2. Customize the template with your preferred structure, sections, and default content
3. Click **"Save"** to store your template

### Step 3: Apply Template

1. When adding new notes, select **"Use Template"**
2. The template automatically populates all fields and sections
3. Start filling in your specific trade or journal entry details

## Benefits

Using templates ensures:

- Consistent note structure across all entries
- Reduced setup time for each new note
- Standardized information capture for better analysis
- Seamless workflow integration
