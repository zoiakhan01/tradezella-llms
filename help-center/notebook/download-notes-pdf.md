# How to Download Your Notes into a PDF File

TradeZella's Notebook feature allows you to export your notes as PDF files. This is perfect for sharing notes with mentors, preserving your work, or reading offline.

## Why Export Notes to PDF?

PDF exports are useful for:

- Sharing your analysis with mentors or trading partners
- Creating backups of important trading notes
- Printing hard copies for your reference
- Offline access without requiring TradeZella login
- Including notes in trading journals or documentation
- Archiving historical trading analysis
- Easy distribution via email

## How to Export a Single Note

To download an individual note as a PDF:

1. Open the Notebook section in TradeZella
2. Find and click on the note you want to export
3. Click the Menu button (three dots) in the note header
4. Select "Export as PDF" or "Download as PDF"
5. The PDF will automatically download to your computer
6. Choose where to save the file when prompted

## How to Export Multiple Notes

To export several notes or an entire folder as PDF:

1. Navigate to the Notebook section
2. Select the folder or multiple notes you want to export
3. Use the "Select All" option if exporting an entire folder
4. Right-click on the selection
5. Choose "Export Selected as PDF"
6. Select whether to create a single file or separate files
7. Confirm the export

## PDF Export Options

When exporting, you can customize:

- **File Format**: Single PDF or separate PDFs per note
- **Included Content**: Text, images, attachments
- **Formatting**: Preserve original formatting or standard layout
- **Cover Page**: Add a cover page with title and date
- **Table of Contents**: Include navigation for multi-page exports

## Customizing Your PDF Export

Before downloading, configure these settings:

1. Select your notes for export
2. Click "Export Options"
3. Choose your preferred settings:
   - Include timestamps and tags
   - Include associated trades (if applicable)
   - Set page orientation (portrait/landscape)
   - Choose font size for readability
4. Click "Preview" to see how your PDF will look
5. Click "Download PDF" to finalize

## Sharing Exported PDFs

Once you've exported your notes as PDF:

- Email the PDF to mentors for review
- Upload to cloud storage for backup
- Share with trading groups or communities
- Print for offline reference
- Store in your trading documentation system

## File Naming and Organization

Exported PDFs are named automatically:

- Single note: "Note_Title_Date.pdf"
- Multiple notes: "Notebook_Export_Date.pdf"
- Folder export: "Folder_Name_Export_Date.pdf"

You can rename files after download for easier organization.
