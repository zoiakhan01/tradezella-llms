# How to Add the Progress Tracker Widget to Your Dashboard

The Progress Tracker widget provides at-a-glance visibility of your trading progress and goals. Adding it to your dashboard ensures you see your performance metrics immediately when you log in.

## Why Add the Progress Tracker Widget?

Benefits of having the widget on your dashboard:

- Monitor daily progress toward your trading goals
- Track multiple metrics simultaneously
- Maintain motivation and accountability
- Quick reference for performance metrics
- Visual representation of goal completion
- Immediate awareness of progress tracking data

## How to Add the Widget

Adding the Progress Tracker widget to your dashboard is simple:

1. Log in to your TradeZella account
2. Navigate to your Dashboard
3. Click "Customize Dashboard" or the "+" button to add widgets
4. Search for "Progress Tracker" in the available widgets list
5. Click "Add Widget" or drag the widget to your dashboard
6. The widget will appear on your dashboard
7. Click "Save" or "Done" to finalize

## Positioning the Widget

Once added, you can customize the widget location:

1. Click and drag the widget to reposition it
2. Place it in a prominent location for easy viewing
3. Resize the widget if options are available (drag corners)
4. Arrange it alongside other important widgets
5. Save your dashboard layout

## What the Widget Displays

The Progress Tracker widget shows:

- **Daily Goal Progress**: Current progress toward today's target
- **Win Rate**: Your trading win percentage
- **P&L Performance**: Daily profit or loss
- **Trade Count**: Number of trades completed
- **Goal Status**: Visual indicator of goal achievement
- **Streak Counter**: Days in a row meeting your goals

## Configuring Widget Settings

To customize what the widget displays:

1. Click the Menu button (three dots) on the widget
2. Select "Widget Settings" or "Configure"
3. Choose which metrics to display:
   - Daily P&L
   - Win rate
   - Trade count
   - Goal progress percentage
   - Streak information
4. Set display preferences (chart type, colors, etc.)
5. Save your preferences

## Multiple Progress Tracker Widgets

You can add multiple Progress Tracker widgets:

1. Each widget can track different metrics
2. Create separate widgets for different trading sessions
3. Compare different time periods side by side
4. Monitor multiple goals simultaneously

For example:
- Widget 1: Daily trading metrics
- Widget 2: Weekly progress toward monthly goal
- Widget 3: Strategy-specific performance tracking

## Removing the Widget

If you want to remove the Progress Tracker widget:

1. Click the Menu button (three dots) on the widget
2. Select "Remove Widget"
3. Confirm the removal
4. The widget will be removed from your dashboard
5. You can always re-add it later

## Refreshing Widget Data

The widget updates automatically:

1. Data refreshes in real-time as you add trades
2. Widget updates when goals are achieved
3. Manual refresh available via widget menu
4. Clear cache if data appears outdated

## Tips for Maximum Effectiveness

- Place the widget in a high-visibility area
- Configure it to show your most important metrics
- Review the widget daily before trading
- Use it to build accountability
- Update your goals regularly for continued motivation

## Widget Compatibility

The Progress Tracker widget works with:

- All TradeZella dashboard layouts
- Mobile and desktop viewing
- Custom dashboard arrangements
- Other dashboard widgets
