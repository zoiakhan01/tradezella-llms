# Dashboard Views in TradeZella

TradeZella offers seven different dashboard views to help you analyze your trading performance from multiple perspectives. Switch between views instantly to get the data presentation that works best for your analysis.

## Available Dashboard Views

### Dollars View (Default)
The default view that displays all dashboard metrics in absolute currency values. Use this view for standard profit/loss tracking and when working with a specific account size.

### Percentage View
Shows all dashboard metrics as percentages relative to your account balance. This view is useful for comparing performance across different account sizes or tracking progress independent of absolute dollar amounts.

### Privacy View
Obscures sensitive account values for screenshots, presentations, or sharing with others. Perfect for using dashboard screenshots in public forums or social media without revealing your account size or specific profits/losses.

### R-Multiple View
Expresses all results in terms of risk units. If you define your risk per trade (your "R"), this view shows how many multiples of that risk each metric represents. Ideal for traders focused on risk-adjusted returns and position sizing methodology.

### Ticks View
A specialized view designed exclusively for futures traders. Displays performance metrics in terms of ticks, the smallest price movement for a futures contract. Use this when your trading strategy is based on tick measurements.

### Pips View
A forex-specific view that displays all metrics in pips (percentage in points), the standard unit for currency pair price movements. Essential for forex traders analyzing their performance in the correct denominations.

### Points View
Another futures-focused view that uses point measurements rather than ticks. Points provide a different scale perspective for futures performance analysis.

## How to Switch Views

Switching between dashboard views is quick and seamless:

1. Click the **View** button located in the top bar of your dashboard
2. Select the desired view from the dropdown menu
3. Your entire dashboard updates instantly to display metrics in the selected view

No data is lost or modified - you're simply viewing the same information in a different format.

## Choosing the Right View

- Use **Dollars** for standard daily tracking
- Use **Percentage** for account-size-independent analysis
- Use **Privacy** when sharing screenshots publicly
- Use **R-Multiple** for risk-focused trading strategies
- Use **Ticks** or **Points** for futures trading
- Use **Pips** for forex trading
