## Data Availability by Symbol

### Historical Data Coverage

- **Forex pairs**: 20+ years of daily data, 10+ years of minute data
- **Futures**: 15-20 years of data (contract-dependent)
- **US Stocks**: 20+ years for most stocks, ETFs
- **Cryptocurrencies**: Full history since inception (Bitcoin: 2009+)
- **Commodities**: 20+ years for futures, commodity indices

### Intraday Data Availability

- **1-minute data**: All symbols (where supported by exchanges)
- **5-minute data**: All symbols
- **15-minute data**: All symbols
- **Hourly data**: All symbols
- **Daily data**: All symbols
- **Weekly/Monthly**: All symbols

## Adding New Symbols

### How to Request New Symbols

If you need a symbol not currently listed:
1. Go to **Settings > Symbols**
2. Click **Request New Symbol**
3. Enter symbol and description
4. TradeZella will notify you when available

### Symbol Update Schedule

- New forex pairs: Monthly
- New stocks: Quarterly
- New futures: As they're launched
- New cryptocurrencies: As they gain liquidity

## Symbol Format

### Entering Symbols Correctly

Standard formats:
- **Forex**: EURUSD, GBPUSD (5-letter pairs)
- **Futures**: ES, NQ, CL (1-3 letter contracts)
- **Stocks**: AAPL, MSFT (all caps)
- **Cryptocurrencies**: BTC, ETH (uppercase abbreviations)
- **ETFs**: SPY, QQQ (all caps)
