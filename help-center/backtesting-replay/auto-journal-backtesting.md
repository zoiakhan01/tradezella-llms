# Do Backtested Trades Get Automatically Journaled in TradeZella?

Yes! TradeZella automatically creates journal entries for every trade you make during backtesting sessions. This seamless integration helps you build a comprehensive trading record without additional manual work.

## How Automatic Journaling Works

### Automatic Entry Creation

When you close a trade in backtesting, TradeZella automatically creates a corresponding journal entry that includes:
- **Entry details**: Symbol, timeframe, date/time, entry price
- **Exit details**: Exit price, exit date/time, position size
- **Calculations**: Profit/loss, R-multiple, win/loss status
- **Chart snapshot**: Visual record of the trade setup
- **Trade context**: Session name and backtesting parameters

### What Gets Recorded

Each auto-journaled backtested trade captures:
1. **Setup identification**: Entry signal and technical setup used
2. **Entry information**: Price, type of order, quantity
3. **Exit information**: Exit price, exit reason, type of exit
4. **Risk/Reward**: Initial risk, target, actual R-multiple achieved
5. **Time in trade**: Duration from entry to exit
6. **Performance metrics**: Individual trade statistics

## Distinguishing Backtested Trades

TradeZella clearly marks backtested trades in your journal:
- **Source tag**: "Backtesting" label on all auto-journaled trades
- **Session reference**: Link to the specific backtesting session
- **Visual indicator**: Distinct styling to separate from live trades
- **Filtering option**: Filter journal to show only backtested trades

## Accessing Auto-Journaled Trades

### In the Journal Section
1. Go to **Journal** in the main navigation
2. Filter by date range or backtesting session
3. Click any trade to view full details
4. Edit or add notes as needed

### From the Backtesting Session
1. Open a completed backtesting session
2. View the **Trade Summary** tab
3. Click on any trade to see its journal entry
4. Add notes directly from the session view
