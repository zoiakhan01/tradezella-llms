# Keyboard Shortcuts in Backtesting

Mastering keyboard shortcuts in TradeZella backtesting dramatically improves your efficiency and trading speed. Here's a comprehensive guide to all available shortcuts.

## Navigation Shortcuts

### Playback Control

| Action | Windows | Mac |
|--------|---------|-----|
| Play/Pause | Spacebar | Spacebar |
| Play faster | Ctrl+Up | Cmd+Up |
| Play slower | Ctrl+Down | Cmd+Down |
| Fast playback (20x) | Ctrl+Shift+Right | Cmd+Shift+Right |
| Slowest playback (1x) | Ctrl+Shift+Left | Cmd+Shift+Left |
| Fastest playback (unlimited) | Ctrl+Shift+Up | Cmd+Shift+Up |


## Customizing Shortcuts

### Creating Custom Shortcuts

1. **Go to Settings**
2. Click **Keyboard Shortcuts**
3. Search for action you want to customize
4. Click **Edit** next to the shortcut
5. Press your desired key combination
6. Click **Save**

### Saving Shortcut Profiles

- Create profiles for different trading styles
- Switch between profiles instantly
- Share profiles with team members
- Import community-created profiles

## Tips for Mastering Shortcuts

1. **Start with essentials**: Learn Play/Pause, Next/Previous, Buy/Sell first
2. **Add one per week**: Learn a new shortcut each week
3. **Use frequently**: Integrate shortcuts into your workflow
4. **Print reference**: Keep shortcut list near your desk
5. **Create muscle memory**: Practice shortcuts repeatedly
6. **Customize for yourself**: Adjust shortcuts to your preference
7. **Use profiles**: Different profiles for different strategies

## Keyboard Shortcut Conflicts

### Windows Conflicts
- Some Windows shortcuts may conflict with system shortcuts
- Customize conflicting shortcuts to avoid issues
- Test after customization

### Mac Conflicts
- Mac system shortcuts take priority
- Remap conflicting shortcuts in settings
- Common conflicts: Cmd+Space, Cmd+Tab
