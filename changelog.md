# TradeZella Changelog

TradeZella ships weekly updates with new features, improvements, and bug fixes. The full changelog is available at [trade-zella.upvoty.com/f/changelog](https://trade-zella.upvoty.com/f/changelog/). Updates are categorized as New, Improved, or Fixed.

## Recent Updates (March 2025)

### March 27, 2025

**Improved:**
- [Backtesting] Economic calendar now displays current date in session and automatically scrolls as session plays
- [Backtesting] Ability to set stop loss and profit target in PIPS (points, ticks, and % coming next)

**Fixed:**
- [Dashboard] Users with no accounts seeing backtesting data on tracking dashboard
- [Trade Page] Tags not displayed after adding and switching tabs
- [Backtesting] Limit order execution timing on seconds timeframe
- [Backtesting] Daily journal page added to backtesting journaling section

### March 21, 2025

**New:**
- [Playbook] First set of Playbook templates now available under the shared playbook tab
- [Backtesting] Support for custom timeframes in backtesting

**Improved:**
- [Backtesting] Added "time spent" column to each session

**Fixed:**
- [Notebook] PDF download issue — text cut off in exports
- [Backtesting] Average calculation on scaling in/out
- [Playbook] Create/edit/view modals broken in mobile view
- [Dashboard] HTTP error 500 when filtering by tag with new widget
- [Tags Management] Error 500 when resetting tags to default
- [Notebook] Image caption not visible in dark mode

### March 14, 2025

**New:**
- New Dashboard widget: "Win%, Avg Win$, Avg Loss$" — view your metrics in a trend
- [Backtesting] Auto breakeven order panel feature — automatically move stop loss to breakeven after price reaches a certain point or RR

**Improved:**
- Updated global button styles
- [View My Day] Changed behavior of "Start my Day" button

## Update Cadence

TradeZella releases updates on a **weekly basis**, typically on Thursdays/Fridays. Each update includes a mix of new features, improvements to existing functionality, and bug fixes across all areas of the platform including Dashboard, Backtesting, Trade Page, Notebook, Reports, and more.
